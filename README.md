 Electronics Price Prediction

A Machine Learning web application that predicts the prices of:

-  Mobile Phones
-  Televisions
-  Air Conditioners

Built using Python, Streamlit, and Scikit-learn.
## Live App

[Open the App](https://electronicspriceprediction-bcywwftrplabck5lxxbbad.streamlit.app/)
---

##  Features

- Multi-category product price prediction
- Interactive Streamlit user interface
- Separate machine learning models for each category
- Instant price estimation

---

##  Technologies Used

- Python
- Streamlit
- Pandas
- Scikit-learn
- Random Forest Regressor
- Pickle
- Git & GitHub
